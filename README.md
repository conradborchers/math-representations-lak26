# Representation Learning for Open Response Analytics

Supplementary repository for the manuscript *"Disentangling Learning from Judgment: Representation Learning for Open Response Analytics,"* accepted as a short paper to LAK '26.

## Folder Structure

* `sampling.ipynb`: Subsetting and embedding data from the `QC23-Complete` data set.
* `main.ipynb`: Analysis code to be run on output of `sampling.ipynb`.
